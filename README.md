# weave-merge

AUR package for [weave](https://github.com/Ataraxy-Labs/weave) — an entity-level semantic merge CLI that resolves conflicts at the function/class level instead of lines.

## Installation

```bash
yay -S weave-merge
```

Or build manually:

```bash
makepkg -si
```

## Included Binaries

- `weave` — core merge tool
- `weave-driver` — git merge driver
- `weave-github` — GitHub integration
- `weave-mcp` — MCP server
